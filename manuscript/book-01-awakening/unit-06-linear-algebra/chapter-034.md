# 第三十四章：坍缩

门后的房间，光很暗。

余路走进去，脚下是同一个三维空间，但这里没有悬满四方的向量了——只有一个孤零零的立方体，浮在房间中央，通体发着微弱的蓝光。立方体的三条棱，正是那三根骨架。

立方体在**缩**。

不是整体变小。是被压。它先是从正方体被压成了一块扁平的板，然后板继续变薄，薄成一张纸，纸继续压，压成一条线，线再收缩……余路屏住呼吸看着，那个曾经饱满的三维立方体，正在一个维度一个维度地丢失自己。

墙上是第六种铅笔字：

> *看好了。这就是"压扁"。*
> *——有一个数，专门量它压了多少。*
> *——第六。*

余路盯着那个正在坍缩的立方体，忽然抓住了那个"数"是什么。

一个变换把空间揉一下，立方体的**体积**会跟着变。有的变换让体积翻倍，有的减半，有的让它膨胀十倍。那个"体积变成原来的几倍"的倍数——就是**行列式**。det。

det = 2，立方体体积翻倍。det = 0.5，体积减半。det = 1，体积不变，空间只是被转了个身或推歪了，但没胀没缩。

那 det = 0 呢？

他看着眼前——立方体正被压成一张纸。纸是二维的，**没有体积**。体积变成了零。这就是 det = 0：变换把三维的立方体压进了一个平面，一整个维度被抹平了。

他翻开笔记本，写道：

> *行列式 det = 体积变化率。变换后，空间体积变成原来的几倍。*
> *det = 2 → 体积翻倍。det = 1 → 体积不变（只是转身或推歪）。*
> *det = 0 → 体积归零，空间被压扁，丢掉了一整个维度。*

---

立方体继续坍缩。纸压成线，线缩成点。最后，整个三维空间——连同它的三根骨架——全部塌进了原点。一个点。

零维。

余路站在空荡荡的房间里，看着那个曾经装满向量、有骨架、会旋转的三维世界，如今只剩下脚边一个孤零零的光点。

他忽然懂了第六种字迹说的"回不来了"。

他试着"倒放"这个变换——把那个点重新拉回立方体。他伸出手，想抓住那个压扁的动作，反着做一遍，把丢掉的维度找回来。

**找不回来。**

问题不在于难。问题在于：当立方体被压成一张纸的那一刻，"高度"这个方向上的所有信息，就**彻底消失了**。原来站在不同高度的两个点，被压扁之后，落到了纸上的同一个位置。现在你拿着这张纸，问"这个点原来在多高的地方"——没有答案。它可能来自任何高度。信息不是"藏起来了"，是**被抹掉了**。

余路的手停在半空。

这就是"不可逆"。一个 det = 0 的变换，没有逆变换——不是因为逆变换太难算，是因为它**根本不存在**。你没法从一张纸，还原出被它压掉的那个维度，就像你没法从一张照片，还原出被拍扁的纵深。丢了就是丢了。

而只要 det 不等于零——哪怕是 0.001，只要不是零——空间就还是那个完整的空间，没有维度被抹掉，你总能把变换倒着做回去。**可逆与不可逆之间，只隔着一个"零"。** 一边是"还能回头"，另一边是"永远回不来"。中间没有过渡地带。

他翻开笔记本，写下：

> *det = 0 → 不可逆。不是"难以逆转"，是"逆变换不存在"。*
> *因为一个维度被压没了，那个维度上的信息被抹掉——两个不同的点落到了同一处，再也分不开。*
> *可逆和不可逆之间，只隔一个零。要么回得来，要么永远回不来，没有中间。*

---

房间安静下来。那个孤零零的光点，在黑暗里轻轻搏动，像一颗还没熄灭的余烬。

墙上的铅笔字，这一次浮现得极慢，一笔一画，淡得几乎要消失。余路必须凑得很近才看得清：

> *我第六世，走得比谁都远。*
> *我几乎摸到了真相——就差最后一步。*
> *然后，有什么东西对我做了一次 det = 0 的变换。*
> *我被压扁了。丢了一个维度。*
> *——我回不来了。这就是为什么，你现在看到的我，只剩铅笔字，淡得快看不见。*
> *——第六。*

余路站在光点旁边，久久没有动。

他一直以为"第六种字迹"只是又一个前世留下的注解，像前面几位一样——公理、素数、弯曲、不变、速记。可这一次不一样。第六个人不是在讲数学。他/她是在用行列式，讲**自己身上发生过的事**：一个走到离真相最近的地方的人，被某种力量"压扁"了，丢掉了一个维度，再也回不到完整的自己。所以留下的痕迹才那么淡——不是不想写清楚，是**他/她本身就已经被抹掉了一部分**。

余路想起"速记"在决斗前夜留下的那句警告的余音，也想起这一路走来，第六种字迹总是躲躲藏藏、用铅笔、淡得快看不见的样子。像在躲什么人。

他掏出笔记本，在空白页上郑重地写下：

> *第六个人——还没有名字，用铅笔，淡得快看不见。近代失踪的数学家。*
> *他/她走得最远，离真相最近，然后"被压扁了，丢了一个维度，回不来了"。*
> *——det = 0。不可逆。*
> *——是谁对他/她做的？为什么？"别让第一种找到你"——警告过我的，是不是就是这件事？*
> *——我要走完他/她没走完的路。但我不能被压扁。我必须守住我的维度。*

他合上笔记本，抬起头。

房间中央那个孤零零的光点，忽然轻轻一颤，重新舒展开来——先是拉成一条线，再撑成一个面，最后重新鼓成一个完整的、饱满的立方体。三根骨架回到了原位。空间恢复了。

不是他做到的。是这个房间自己"重置"了，像一段被倒回去重放的录像。可余路知道，真实世界里没有这样的重放。第六个人不会重新鼓起来。det = 0 的那一下，是真的。

立方体安静地浮在房间中央，重新开始缓慢地旋转——不是坍缩，是**旋转**。绕着一根看不见的轴。

余路的目光落在那根轴上。在整个立方体转动的时候，唯独那根轴的方向，**一动不动**。

墙上，第六种铅笔字浮起最后一行，这一次不再是悲伤的语气，倒像是在把最要紧的东西交到他手里：

> *别怕。就算空间被转得天翻地覆——*
> *总有一个方向，是不会变的。*
> *找到它。站上去。*
> *——第六。*

房间尽头，一道门打开了。门后有光——不是暗光，是稳定的、不闪动的白光。

余路朝那根不变的轴看了最后一眼，走了过去。

---

*（第三十四章完）*
