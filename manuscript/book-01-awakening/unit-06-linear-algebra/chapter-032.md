# 第三十二章：城的骨架

门后又是一个三维空间，和刚才的立方体几乎一样——同样的原点，同样悬满四方的向量。

但这一次，空间的正中央，立着**三根光柱**。

三根光柱都从原点出发。第一根笔直地沿地板向前伸，第二根沿地板向右，第三根笔直向上，戳向天花板。它们互相垂直，像一个被撑开的三维坐标架，把整个空间钉在原点上。

墙上浮着第六种铅笔字：

> *这三根，就是这个空间的骨架。*
> *——第六。*

余路走到三根光柱之间。他伸手碰了碰向前的那一根，它微微一颤，发出低低的嗡鸣。他明白这是什么——这是三个**基向量**：(1,0,0)、(0,1,0)、(0,0,1)。空间的三个基本方向。

他想验证"骨架"这个说法。他随手在空间里选了一个向量——一条指向斜上方的箭头，坐标大概是 (2, 3, 1)。他盯着它，问自己：这条箭头，能不能只用那三根光柱"拼"出来？

能。

他几乎是立刻就"看"到了：沿第一根光柱走 2 步，再沿第二根走 3 步，再沿第三根走 1 步——正好到达那条箭头的尖端。(2,3,1) = 2×(1,0,0) + 3×(0,1,0) + 1×(0,0,1)。三根光柱，各按一个倍数取来，加在一起，就拼出了那个向量。

他又随手选了别的向量。每一个都能拼。无论箭头指向哪里，多长，多歪，他总能算出"沿三根光柱各走多少步"，把它拼出来。

余路慢慢明白了"骨架"的分量。

整个三维空间，无穷多个向量，密密麻麻，看起来杂乱无章——可其实**它们全都是这三根光柱的组合**。你不需要记住无穷多个向量。你只需要记住这三根。剩下的一切，都能用这三根"搭"出来。这三根光柱，就是空间的**基底**。

他翻开笔记本，写道：

> *基底 = 空间的骨架。*
> *空间里有无穷多个向量，但它们全是几根基向量的组合。*
> *记住骨架，就等于记住了整个空间。*

---

一个问题冒了出来：为什么是**三**根？

余路盯着那三根光柱。他试着去掉一根——用手"掐灭"了向上的那根。空间立刻塌了：所有原本需要"向上"分量的向量，现在都拼不出来了。天花板方向的整个世界消失了，空间从三维塌成了平铺在地板上的二维。

他把那根光柱重新点亮，三维空间恢复了。

那……能不能反过来，**多加**一根？他试着在空间里又拉出第四根光柱，让它指向 (1, 1, 0) 的方向。

墙上的铅笔字动了：

> *第四根，是多余的。*
> *——第六。*

余路一愣，随即懂了。第四根光柱指向 (1,1,0)——可这个方向，用前两根光柱就能拼出来：(1,1,0) = 1×(1,0,0) + 1×(0,1,0)。它没有带来任何"新方向"。它是**多余**的。加上它，空间不会变大一分；去掉它，空间也不会缺一块。

他掐灭了那根多余的光柱。

于是他触到了两个彼此咬合的概念。第一个：一组基向量必须**谁也不能被其他几根拼出来**——每一根都提供一个别人给不了的新方向。这叫**线性无关**。三根光柱里，任意一根都不是另两根的组合，所以它们线性无关；刚才那第四根能被拼出来，所以它是"相关"的，是多余的。

第二个：撑起这个空间，**最少需要几根**这样谁也替不了谁的骨架？三根。少一根就塌，多一根就多余。这个数——三——就是空间的**维数**。

余路站在三根光柱中间，忽然觉得整件事变得异常干净。

一个空间，无论装了多少向量，看起来多么庞大复杂，它的"复杂度"其实只由一个数决定——维数。三维空间，本质上就是"三根骨架撑起来的世界"。你只要抓住那三根线性无关的骨架，就抓住了整个空间的全部——一根不能少，一根不必多。

他闭上眼，用了"破形者"的感知，去看那些"可以当向量的别的东西"。

多项式。所有次数不超过二的多项式——ax² + bx + c——它们也构成一个向量空间。这个空间的骨架是什么？他"看"到了：1、x、x²。任何这样的多项式，都是这三样东西的组合：ax² + bx + c = a×(x²) + b×(x) + c×(1)。三根骨架。所以"二次以下多项式"的空间，也是**三维**的。

它和刚才那个装箭头的立方体，长得完全不一样——一个是几何箭头，一个是代数式子——可它们的**骨架数目一样**，都是三。在"维数"这件事上，它们是同一个空间。

余路睁开眼，心里那点熟悉的震颤又来了。他想起破形者的直觉：不同的结构，可以有同一副骨架。此刻在这里，它有了一个精确的形状——**维数相同的向量空间，本质上是同一个空间**，无论它们表面上装的是箭头、多项式，还是别的什么。

他翻开笔记本，写道：

> *线性无关 = 每根骨架都提供一个别人给不了的新方向，谁也拼不出谁。*
> *维数 = 撑起空间，最少需要几根线性无关的骨架。少一根就塌，多一根就多余。*
> *——空间再大再乱，复杂度只由"维数"这一个数决定。抓住骨架，就抓住了整个空间。*
> *——装箭头的三维空间，和"二次以下多项式"的空间，骨架都是三根。维数相同 = 本质相同。*

他合上笔记本。

三根光柱静静地立在原点上，稳稳地撑着整个空间。它们不动——至少在这个房间里，它们不动。

但余路记得，这座城是会**动**的。从外面看进来，每一栋建筑都在拉伸、剪切、旋转。那么当整个空间开始变换的时候，这三根骨架，会发生什么？

空间的一侧，一道门缓缓打开。门外传来一阵低沉的、机械般的嗡鸣——像是有什么庞大的东西，正准备转动起来。

余路朝那道门走去。

*（第三十二章完）*
