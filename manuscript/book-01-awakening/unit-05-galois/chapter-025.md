# 第二十五章：五次方程

伽罗瓦纪念馆是一座废弃的数学图书馆。

余路根据"不变"留下的线索，在城市的另一端找到了它。图书馆坐落在一条安静的林荫道上，外观是古典的欧式建筑——石柱，穹顶，铜门。门上刻着第五种字迹的符号。符号旁边有一行小字，第五种字迹——那种潦草到接近符号的——第一次写了完整的句子：

> *五次方程。x⁵ + ax⁴ + bx³ + cx² + dx + e = 0。没有根式解。*
> *——不是"暂时没找到"。是"不存在"。*
> *——第五。*

余路推开了门。

---

图书馆的大厅是一个高挑的空间，穹顶是玻璃的，阳光从上方洒下来，照亮了大厅中央的五个石碑。五个石碑并排矗立，从左到右，每一个上面都刻着一个多项式方程。

第一个石碑：x + a = 0。一次方程。旁边刻着解：x = -a。

第二个石碑：x² + ax + b = 0。二次方程。旁边刻着解：x = (-a ± √(a² - 4b)) / 2。

第三个石碑：x³ + ax² + bx + c = 0。三次方程。旁边刻着解——卡尔达诺公式，一个复杂的、包含立方根和平方根的表达式。

第四个石碑：x⁴ + ax³ + bx² + cx + d = 0。四次方程。旁边刻着解——费拉里公式。

第五个石碑：x⁵ + ax⁴ + bx³ + cx² + dx + e = 0。五次方程。旁边**没有解**。只有一行小字，第五种字迹：

> *阿贝尔-鲁菲尼定理（1824年）：五次及以上的多项式方程，没有通用的根式解公式。*
> *不是"暂时没找到"——是"不存在"。*
> *——第五。*

余路站在第五个石碑前。他当然知道阿贝尔-鲁菲尼定理——任何学过抽象代数的人都知道。但他从来没有真正**理解**过它。他理解的是"定理的内容"——"五次方程没有根式解"。他从来不理解的是**为什么**。一次到四次都可以，五次就不行？四次和五次之间发生了什么？数学在四次和五次之间，**断裂**了。

他用了"破形者"的感知。他看到了前四个石碑中根的排列形成的置换群。一次方程——一个根，群是平凡的。二次——两个根，S₂，可解。三次——三个根，S₃，可解。四次——四个根，S₄，可解。但五次方程有五个根——一百二十种排列，S₅。**不可解**。

他忽然理解了。不是"五次方程太难了"——是**五次方程的对称群S₅在结构上不可解**。S₅有一个正规子群A₅（60个元素，所有偶置换），A₅是**单群**——没有非平凡的正规子群。在伽罗瓦理论中，方程有根式解的充要条件是它的伽罗瓦群是**可解群**。可解群需要一个"正规子群链"从群本身一直降到平凡群。S₅ → A₅ → ???。A₅是单群，链断了。所以S₅不可解。所以五次方程没有根式解。

他翻开笔记本，写道：

> *五次方程没有根式解——不是因为"太难"，而是因为S₅不可解。*
> *S₅ → A₅ → ???。A₅是单群，链断了。结构约束了可能性。*
> *——认知也是一样。有些认知问题，在你的认知模式中"不可解"——不是因为你不够聪明，而是因为你的认知结构约束了答案。*
> *要"解"它，你需要换一个认知结构——换一个群。*

---

大厅的墙上挂着一排手绘素描。第一幅——尼尔斯·阿贝尔。1824年证明五次方程不可解。二十六岁死于肺结核。第二幅——埃瓦里斯特·伽罗瓦。1832年——决斗前夜——写下了伽罗瓦理论。二十一岁死于决斗。

他站在伽罗瓦的肖像前。肖像中的伽罗瓦是一个年轻人，眼神锐利，嘴唇紧抿。他看起来不像"数学家"——更像**革命者**。他的父亲是市长，在政治斗争中自杀。他因参加共和派活动被学校开除，因政治言论被捕入狱。他的数学论文被柯西弄丢了，被傅里叶弄丢了，被泊松退回了。他的生命在决斗的前夜——他只有二十一岁——达到了顶峰。

第五种字迹在肖像下方写道：

> *伽罗瓦在决斗前夜写了一封信。信中说："我没有时间了。我没有时间了。但我的思想——请把它们寄给高斯，寄给雅可比。他们会理解的。"*
> *高斯没有收到。雅可比没有收到。伽罗瓦的思想在档案室里躺了十四年，直到刘维尔在1846年发现了它们。*
> *——他死了。但他的理论活了两百年。*
> *——第五。*

余路站在肖像前，忽然感到一种奇异的情绪——不是悲伤，不是愤怒。是**认可**。一个二十一岁的年轻人，在生命最后一夜，把所有的思想倾泻在纸上——不是因为他想要"被记住"，而是因为他**不能让这些思想和他一起死**。他必须把它们写下来。不是"想要"——是**必须**。

他忽然想到了自己。他也在"写"——他的笔记本上已经写满了数学概念的可视化、认知模式的感悟。他也是在"写"——不是因为他想要"被记住"，而是因为如果他在这一世失败了——就像前六世一样——他需要留下线索给第八世。就像伽罗瓦在决斗前夜留下线索给后来的数学家。

他掏出笔记本，写道：

> *伽罗瓦在二十一岁死了。但他留下的东西，改变了数学。*
> *"对称性决定可解性"——不仅是方程，也是人生。*
> *你的群是什么？你的群可解吗？如果不可解，不是"你不够努力"——是"你需要换一个群"。*

他转身朝大厅深处走去。那里有一扇门，门上方刻着第五种字迹的符号——圆里有三角形，三角形被五个小球包围，排列方式不断变换。

他推开了门。
