# 第二十八章：五次方程

伽罗瓦纪念馆是一座废弃的数学图书馆。

余路根据"不变"留下的线索，在城市的另一端找到了它。图书馆坐落在一条安静的林荫道上，外观是古典的欧式建筑——石柱，穹顶，铜门。门上刻着第五种字迹的符号。符号旁边有一行小字，第五种字迹——那种潦草到接近符号的——第一次写了完整的句子：

> *五次方程。x⁵ + ax⁴ + bx³ + cx² + dx + e = 0。没有根式解。*
> *——不是"暂时没找到"。是"不存在"。*
> *——第五。*

余路把手放在铜门上。门很凉。他站了一会儿，才推开它。

---

图书馆的大厅是一个高挑的空间，穹顶是玻璃的，阳光从上方洒下来，照亮了大厅中央的五个石碑。五个石碑并排矗立，从左到右，每一个上面都刻着一个多项式方程。

第一个石碑：x + a = 0。一次方程。旁边刻着解：x = -a。

第二个石碑：x² + ax + b = 0。二次方程。旁边刻着解：x = (-a ± √(a² - 4b)) / 2。他认得这个——每个中学生都认得这个。古巴比伦人在四千年前就知道它的雏形。

第三个石碑：x³ + ax² + bx + c = 0。三次方程。旁边刻着解——卡尔达诺公式，一个复杂的、包含立方根和平方根的表达式，一层套着一层，像一座用根号搭起来的塔。十六世纪。

第四个石碑：x⁴ + ax³ + bx² + cx + d = 0。四次方程。旁边刻着解——费拉里公式。更复杂，更长，但依然是**根号搭起来的塔**。依然只用加、减、乘、除、开方。同样是十六世纪。

余路沿着石碑走。他能感觉到一种节奏——从一次到四次，公式在变长，变复杂，但它们是**同一类东西**。都是有限次的加减乘除开方。都是"根式"。像是四级台阶，一级比一级高，但都是台阶，都可以踩上去。

然后他走到了第五个石碑前。

第五个石碑：x⁵ + ax⁴ + bx³ + cx² + dx + e = 0。五次方程。

旁边**没有解**。

石碑的右半边是空的——不是磨损了，不是被人凿掉了，是**从一开始就没有刻**。光滑的石面，什么都没有。只有最下方一行小字，第五种字迹：

> *到这里，台阶断了。*
> *——第五。*

---

余路站在第五个石碑前。

他当然知道这件事。任何学过抽象代数的人都知道——五次方程没有一般的根式解。这是教科书里的一个定理，一个名字，一段可以背下来的结论。他"知道"它，就像知道一条街的名字，却从来没有走过那条街。

他试着走一走。

他闭上眼，用"破形者"的感知去够第五个石碑里的东西。前四个石碑，他一够就有回应——根像五颗悬浮的光点，被一层层根号包裹着，他能顺着根号一层层往里剥，最后剥到根。台阶是实的，踩得上去。

第五个石碑不一样。

五颗光点还在那里——五次方程当然有五个根，这是代数基本定理保证的，复数域里一定有。他看得见它们。但当他伸手去够那层"根号搭起来的塔"时——**够不到**。不是塔太高。是**塔不在那里**。

他往前伸手，手穿过了空气。他换了个方向，再伸手，还是空气。他知道根就在那里，五颗光点悬在他面前，触手可及——可是从"系数"到"根"的那条用加减乘除开方铺成的路，**不存在**。

这不是"他找不到"的那种感觉。他找过很多东西，找不到的时候，总有一种"它藏在某处"的余味——你知道它在，只是没找到角度。

这一次没有余味。

他伸手的地方是空的，而那种空不是"藏起来了"的空，是**"这里本来就没有"**的空。像是伸手去够一个房间里并不存在的第四面墙。

余路睁开眼，退后一步。

他的心跳有点快。不是因为害怕，是因为**困惑**。前四个石碑那么顺理成章——一次、二次、三次、四次，公式一个比一个长，但一路走下来毫无阻碍。到第五个，路就这么没了。

为什么？

四次和五次之间，隔着什么？只多了一个 x⁵。只多了一个根。从四颗光点到五颗光点。为什么多这一颗，整条路就塌了？

他盯着那个空白的石面。他能确认"路不存在"——他的感知刚刚亲自撞过那堵空。但他**不知道为什么**。他理解的是"结论"——五次方程没有根式解。他不理解的是"原因"——凭什么？在四和五之间，数学到底发生了什么，让一条走了四千年的路，突然断掉？

他翻开笔记本，在空白页上写下：

> *五次方程没有根式解。我一直"知道"这句话。今天我第一次"撞"到它。*
> *前四个石碑的路是实的。第五个石碑，路是空的——不是藏起来，是不存在。*
> *可是为什么？多一个根，路就塌了？四和五之间，发生了什么？*
> *——我知道结论。我不知道原因。今天之前，我以为这是一回事。*

他合上笔记本。空白的石面在阳光下泛着淡淡的光，安静得像一个不肯回答的人。

大厅的尽头，有一道拱门，门后是另一个房间。墙上似乎挂着什么——从这里看过去，像是几幅画像。

余路朝那道拱门走去。

---

*（第二十八章完）*
