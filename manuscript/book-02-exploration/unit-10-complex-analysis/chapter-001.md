# 第一章：转了九十度的星空

废弃天文馆藏在城市东北角的一片荒地里。

穹顶是灰蒙蒙的，玻璃蒙着厚灰，从外面看像一只倒扣的巨碗。余路推开生锈的铁门，走进去。里面是黑的——穹顶没有光，投影仪沉默着，控制台上落满了灰尘。

他找到电源，按下开关。嗡嗡的电流声响起，穹顶亮了。

投出来的不是星空。

是一片**流动的平面**——像一块巨大的、铺在穹顶内壁上的绸缎，绸缎上布满了密密麻麻的点，每个点都标着一个数字。有的点是实数：1, 2, 3, -1, 0.5。有的点是……别的东西。0+i, 1-i, -2+3i。

余路认出了它。这是**复平面**。横轴是实部，纵轴是虚部。每一个复数 a+bi，对应平面上的一个点 (a, b)。

---

他走到控制台前。台上有几个旋钮，标着不同的符号。他试着转了一下标着"1"的旋钮——平面上的点微微移动，像被推了一下。转标着"2"的旋钮——点被推得更远。数字在"缩放"。

然后他看到了标着 **i** 的旋钮。

他转了一下。

整片星图**逆时针转了 90 度**。

他愣住了。再转一下——又转 90 度。转四下——回到原位。360 度。

他盯着那个标着 i 的旋钮，心跳快起来。

i——虚数单位。他在中学时学过：i² = -1。当时他觉得这只是一个"定义"，一个为了让方程有解而硬造出来的符号。可此刻他看到了 i 的**几何意义**：**乘以 i，就是把整个平面逆时针转 90 度。**

i² = 转 180 度 = -1。i³ = 转 270 度 = -i。i⁴ = 转 360 度 = 1。不是"硬造"——是**旋转**。

他翻开笔记本，写道：

> *i 不是"虚构的数"。i 是一个动作：逆时针旋转 90 度。*
> *乘以 i = 转 90°。i² = 转 180° = -1。i⁴ = 转 360° = 1。*
> *——"虚数"第一次变成一个真实的动作。*

---

他继续探索控制台。

他发现，旋钮不只是 i 和实数。还有一个标着 **e^{iθ}** 的滑块。他推动滑块，θ 的值从 0 开始慢慢增大。

星图开始**匀速旋转**。

不是跳跃式地转 90 度——是**连续地、平滑地**旋转。θ 从 0 增大到 π/2，星图转了 90 度。θ 增大到 π，星图转了 180 度——翻到了背面。θ 继续增大到 2π，星图转了一整圈，回到原位。

**e^{iθ} = cos θ + i sin θ。**

这就是欧拉公式。e 的"虚数次方"，不再是一个抽象的符号——它是一个**旋转操作**。e^{iθ} 把整个复平面旋转 θ 弧度。模长始终是 1（不缩放），方向随 θ 连续变化。

余路把滑块推到 θ = π。

星图旋转 180 度，翻到了背面。穹顶上的数字全部取了相反数——1 变成 -1，2 变成 -2，i 变成 -i。

他忽然想起一个恒等式：**e^{iπ} + 1 = 0**。

这个恒等式把五个最重要的数学常数缝在了一起：e（自然增长）、i（旋转）、π（圆）、1（乘法单位）、0（加法单位）。他当年只是记住了这个公式，觉得它"优雅"。此刻他看到了它的**几何意义**：旋转 180 度（e^{iπ}），正好把 1 变成 -1，所以 e^{iπ} = -1，移项得 e^{iπ} + 1 = 0。

不是"优雅"。是**必然**。旋转半圈，正面变反面——这不是巧合，是几何的必然。

他翻开笔记本，写道：

> *e^{iθ} = cosθ + i sinθ。欧拉公式 = 旋转。e^{iθ} 把复平面旋转 θ 弧度。*
> *e^{iπ} + 1 = 0：旋转半圈，1 变 -1。五个常数缝在一起，不是巧合，是几何的必然。*
> *——乘以一个复数 = 旋转 + 缩放。旋转的角度 = 辐角，缩放的倍数 = 模长。*

---

他在控制台前坐了很久，慢慢地转动各种旋钮，看着星图旋转、缩放、扭曲，又恢复原样。复数不再是抽象的符号——它是**平面上的动作**。每一个复数，都是一个"旋转+缩放"的操作。

然后他注意到了穹顶的正中央——一个亮点。

亮点不大，但很亮，发着白光。他试着靠近它——每走一步，头就晕一点。走到离它大约三步远的时候，他几乎站不稳了，头晕目眩，像被什么东西从里面搅动着。

他赶紧退回来。

那个亮点，在复平面上的坐标，他隐约能看到——是一个特殊的点，一个函数在那里**没有定义**、或者趋向无穷大的点。

**奇点。**

穹顶中央的亮点，安安静静地发着光，像一只不动的眼睛。它不让他靠近——但它就在那里，等着他去理解。

墙上浮起第七种字迹：

> *穹顶中央那个亮点——奇点。靠近就头晕，因为它趋向无穷。*
> *——别急着冲进去。先学会"只读边界"。*
> *——第七。*

余路看了看那个亮点，又看了看穹顶边缘那一圈完整的、连续的光——边界。

他朝边界走去。

*（第一章完）*
