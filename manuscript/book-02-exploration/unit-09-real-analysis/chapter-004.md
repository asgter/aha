# 第四章：几乎处处

余路拿着那沓空白凭证，继续在城里走。

他已经学会了给"正常的"街道发凭证——那些横切后能满足可测条件的集合。可他发现，光有"长度"还不够。他还需要量**更复杂的东西**——不是一条街道的长度，而是一整片街区的"体积"，或者一个在街区上空起伏的曲面的"积分"。

为此，他需要一种新的工具，来处理那些"大部分地方乖、个别地方疯"的函数。

---

老测量员带他走上一座天桥，俯瞰脚下的街区。

"你看，"他说，指着下面一条街道，"这条街，大部分地方是平的——高度为 0。但有几个点，突然是 1。极少数几个点。"

余路顺着他的手看去。街道确实大部分平坦，偶尔有一个点冒出来，像地面上的一根刺。

"这几个'刺'，有理数的位置——和狄利克雷路上的地砖一样。但这里，'刺'的数量少得多——只有可数多个。"

"有理数是可数的，"余路说，"所以这些'刺'的总长度——是 0。"

"对。那这条街的积分是多少？"

余路想了想。"值=1"的点，只占了长度 0。"值=0"的点，占了几乎所有位置。所以积分 = 0 × 1 + 1 × 0 = 0。

"和狄利克雷函数的积分一样。"

"但有一个区别，"老测量员说，"狄利克雷函数在**每一点都不连续**——它疯得彻底。而这条街——只有几个点是'刺'，其他地方都是平的。它**几乎**是连续的。"

"几乎处处"——除了一个测度为 0 的集合外，都成立。这条街"几乎处处"是平的。狄利克雷函数"几乎处处"等于 0。"几乎处处"这个概念，允许你**忽略那些测度为 0 的例外**——因为它们在积分的眼中不存在。

他翻开笔记本，写道：

> *"几乎处处"（a.e.）= 除了一个测度为 0 的集合外都成立。*
> *测度为 0 的集合，可以忽略——因为它在积分中贡献为 0。*
> *——狄利克雷函数"几乎处处"等于 0，所以它的积分是 0。*
> *——"忽略测度为 0 的例外"不是偷懒，是发现"零测集在积分眼中不存在"。*

---

老测量员从天桥上走下来，带他走进一间工坊。

工坊的墙上，挂着一排排"台阶函数"——每一个都是由有限个水平台阶组成的函数图像：第一级台阶高度 0.3，第二级 0.7，第三级 0.9……台阶数有多有少，高度参差不齐。

"这些叫**简单函数**，"老测量员说，"有限个台阶，每一个台阶的底都是可测集。它们是勒贝格积分的'积木'。"

他拿起一个台阶很少的简单函数，放在一个复杂的、起伏的曲面旁边。

"你想量这个曲面的体积——直接量太难。但你可以**用简单函数去逼近它**。先拿一个台阶很少的，粗粗地贴上去。再拿一个台阶多一点的，贴得更紧。再拿一个台阶更密的……台阶越密，逼近越好。"

余路看着那些简单函数一级一级地往上叠，像无穷多级台阶在递增，台阶的顶边越来越贴合那个曲面的轮廓——从粗糙到精细，从棱角到圆润。

"任何非负的可测函数，"老测量员说，"都可以被这样一组递增的简单函数逼近到任意精度。台阶无穷多的时候，逼近就到了极限。"

余路翻开笔记本，写道：

> *简单函数 = 有限个台阶，每个台阶的底都是可测集。勒贝格积分的"积木"。*
> *任何非负可测函数 = 一列递增简单函数的极限。台阶越密，逼近越好。*
> *可测函数 = 横着切时每一层的底都可测的函数——保证逼近的每一步都有意义。*

---

他走到那几条问号街的路口，试着用简单函数去逼近那些"不可测"的街道。

台阶一级一级往上叠——但叠到某处，底就塌了。那一层的底，是不可测集，没有长度，没法算面积。台阶叠不上去。

他退后一步。不可测集的存在，不只是"量不出长度"——它还会传染：任何依赖"横着切每一层的底"的方法，在不可测集面前都会失效。简单函数逼近不了它，因为它的"底"不存在。

"所以，"他低声说，"不可测集不只是地图上的一个问号。它是整个积分体系的一个**盲区**。"

老测量员点了点头。

"你看到了。可测函数的世界，是'横着切'的世界。而那些不可测的街道，站在这个世界之外——它们不属于'横着切'能处理的范围。"

余路合上笔记本，看了一眼那些问号街。它们安安静静的，不参与任何逼近，不被任何简单函数覆盖，像被整个测量体系排除在外的孤岛。

远处的天空，忽然暗了下来。一阵低沉的嗡鸣声，从城市深处传来——像有什么巨大的东西正在逼近。

老测量员抬起头，看了看天色。

"逼近风暴要来了，"他说，"你学过怎么安全地逼近。但风暴里的逼近——是无穷多个逼近真值的影子同时涌来，疯狂震荡。你扛得住吗？"

---

*（第四章完）*
