# 单元 1 研究笔记：数学公理与集合论

## 核心目标
让读者通过主角的眼睛"看见"：
1. 公理不是真理，是选择
2. 集合论的诞生与危机（罗素悖论）
3. 无穷有不同大小（康托尔对角线论证）
4. 选择公理的直觉与影响

---

## 数学概念清单

### 公理系统
- **公理**：不加证明就被接受为真的初始命题
- **一致性**：系统内不会推出矛盾
- **完备性**：所有真命题都可以在系统内被证明
- **独立性**：一个公理不能被其他公理推导出来

**关键历史**：欧几里得第五公设的独立性争论持续两千年。非欧几何的发现证明：第五公设不是"必然真理"，而是"可以选择的前提"。

### 集合论基础
- 集合：一组确定对象的全体
- 元素、子集、幂集
- 并、交、补运算
- 无穷集合：可数无穷 vs 不可数无穷

### 罗素悖论（1901）
定义 R = { x | x ∉ x }（所有不包含自身的集合的集合）。问 R 是否包含自身？两种答案都导致矛盾。

**直觉**：就像理发师悖论——村里的理发师给所有不自己刮脸的人刮脸，他给自己刮吗？这不是文字游戏，是逻辑结构中真实存在的裂缝。

### 康托尔对角线论证（1891）
**结论**：实数比自然数"更多"——存在不同大小的无穷。

**论证直觉**：
1. 假设所有 0-1 之间实数可以列成表（与自然数一一对应）
2. 构造新数：小数点后第 n 位 = 表中第 n 行的第 n 位 + 1
3. 这个新数与表中每个数都至少有一位不同
4. 所以它不在表中——矛盾
5. 因此实数不可数

### 连续统假设
是否存在严格在自然数与实数之间的基数？这个问题在 ZFC 公理系统中既不能被证明也不能被否证——又是"独立的"。

### 选择公理
**陈述**：对于任意一族非空集合，存在一个选择函数从每个集合选一个元素。

**直觉**：有限个盒子没问题；无限个盒子——选择本身变成了需要被"创造"的动作。

**巴拿赫-塔斯基悖论**：在 3D 空间，一个球可被分割为有限块并重新组成两个与原球大小相同的球。依赖于选择公理。

## 可视化想法

| 概念 | 画面 | 主角体验 |
|------|------|---------|
| 公理系统 | 大楼地基——地基下是一系列"被放入"的柱子 | 能看见别人话语下面的柱子材质 |
| 罗素悖论 | 咬自己尾巴的蛇，无限循环 | 思维"打结"——无法闭合的反馈回路 |
| 对角线论证 | 无限长的表，总能构造表外新数 | 意识到有些东西逃逸任何枚举 |
| 选择公理 | 无限个盒子，各取一件，但没有取法规则 | 必须"选"——没有正确选择，只有他的选择 |

## 遗迹设计
- **场景**：旧书店（"回声书屋"）地下密室
- **结构**：无数抽屉（集合定义）→ 无限长桌（实数表）→ 无限个盒子（选择公理）
- **锁的本质**：不是找正确抽屉，而是意识到不存在"正确"，只有"你选的"
