# 单元 3 研究笔记：几何学（从欧氏到非欧）

## 核心目标
让读者通过主角的眼睛"看见"：
1. 第五公设不是"显然的真理"——两千年的证明失败史
2. 非欧几何——空间可以是弯曲的，三角形内角和可以不是180度
3. 高斯曲率（绝妙定理）——弯曲是内禀的，不需要跳出维度就能感知
4. 黎曼流形——每一点都可以有自己的局部几何

---

## 数学概念清单

### 欧几里得第五公设
**原版**：一直线与两直线相交，若同侧内角之和小于两直角，则这两直线在该侧延长后相交。

**等价版（普莱费尔公理）**：过直线外一点，至多有一条直线与该直线平行。

**直觉**：前四条公设是"显然的"——从任一点到任一点可以作直线，所有直角都相等……第五条不是。它读起来像定理，不像公理。两千年来，无数数学家试图从其他四条公设推导出它——全部失败。非欧几何的发现证明：第五公设是**独立的**——它可以被替换。

### 非欧几何
**双曲几何**：过直线外一点，至少存在两条直线与该直线平行。空间是"马鞍形"——三角形内角和 < 180°。

**椭圆几何**：过直线外一点，不存在与该直线平行的直线。空间是"球形"——三角形内角和 > 180°。球面上，从北极→赤道→赤道→北极，三个角都是直角，内角和 = 270°。

### 高斯曲率（Theorema Egregium）
**直觉**：二维生物不需要跳出三维就能测量自己所在曲面的曲率。在曲面上画一个三角形，测量内角和——>180°是正曲率（球面），<180°是负曲率（马鞍面），=180°是平坦的。

**绝妙之处**：曲率是**内禀的**——不依赖于曲面如何嵌入更高维空间。纸卷成圆筒，三角形内角和还是 180°——圆筒的曲率是 0，它是平坦的。

### 测地线
弯曲空间中的"直线"——球面上的大圆是测地线。北京飞纽约的飞机不是沿纬线直飞，而是沿大圆飞——那是"最短路径"。

### 平行移动
在弯曲空间中，把一个向量沿闭合路径平行移动一圈——回到起点时，向量方向变了。曲率改变了"平行"的含义。

### 黎曼流形
每一点都有自己的局部几何。在每一点附近，空间看起来像欧氏空间（平坦的），但在大尺度上，空间是弯曲的。球面在每一点附近看起来像平面，但整体是球面。

## 可视化想法

| 概念 | 画面 | 主角的体验 |
|------|------|----------|
| 第五公设 | 五次尝试画平行线的相框，每次都失败 | 镜廊中看到无数个"失败的平行线" |
| 非欧几何 | 同一扇门，不同人看到不同形状 | 镜中看到不同几何版本的自己 |
| 高斯曲率 | 三角形内角和随位置变化 | 在弯曲空间中画三角形测量内角和 |
| 测地线 | 弯曲空间中的"最直路径" | 在弯曲空间中寻找最短路径 |
| 平行移动 | 向量沿闭合路径走一圈，方向变了 | 镜廊中走一圈，回到起点时"方向"变了 |

## 遗迹设计
- **场景**：废弃镜廊博物馆——"镜中几何"
- **结构**：入口走廊（欧氏）→ 第一镜厅（双曲）→ 第二镜厅（椭圆）→ 第三镜厅（高斯曲率）→ 第四镜厅（黎曼流形）→ 最深处的镜子（无反射，是入口）
- **锁的本质**：镜子不是反射面——它们是**几何公理的"选择器"**。每一面镜子选择了不同的第五公设
- **突破条件**：在弯曲空间中行走，用身体感知"平行"的不同含义，理解曲率是内禀的
